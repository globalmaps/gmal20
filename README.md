## metadata
This Data was developed based on Global Map Specifications version 2.2. 

### Boundary, Transportation, Drainage, Population center
The data source was a Digital Map 1:25000 and Orthophoto, map image made in 1985 and orthophoto made in 2008.

### Elevation
The Global Map Albania version 2 Elevation layer was developed based on Global Map Specifications version 2.2. The data source was developed using orthophoto 2000 year.
 
### Land use, Vegetation
The Global Map Albania version 2 Vegetation layer was developed based on Global Map Specifications version 2.2. The data source was developed using orthophoto 2012 year.
 
### Land Cover
The Global Map Albania version 2 Land Cover layer was developed based on Global Map Specifications version 2.2. IRS, SPOT and RapidEye satellite images, dual coverage, orthophotos (2012), topographic maps for making Land Cover.
